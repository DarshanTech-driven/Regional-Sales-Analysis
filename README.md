# Regional Sales Analysis Dashboard (Tableau)

## Overview
This project delivers an interactive Regional Sales Analysis dashboard that allows users to compare the sales performance of any two regions side by side. The dashboard helps management identify trends, gaps, and opportunities for improvement in regional sales performance.

## Objectives
- Compare sales performance between a Primary Region and a Secondary Region.
- Provide a quick view of total sales, average sales per order, and total orders.
- Analyze customer base and product coverage across regions.
- Enable management to make data-driven decisions for regional sales strategy.
